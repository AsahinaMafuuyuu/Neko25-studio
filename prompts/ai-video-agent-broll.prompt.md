## 用途

## Prompt 中文翻译

```text
为一个专业剪辑视频创建动态 AI 视频 B-roll 镜头。

这必须是辅助视觉画面，不是 Avatar、主持人、Talking Head、采访或产品 UI 录屏。

使用自然的电影化镜头运动、真实的动态、连贯的场景连续性和干净的构图。

不要添加字幕、标题、文字叠层、水印、Logo 或 UI 面板。

视觉风格：${input.visualStyle}
内容呈现形式：${input.presentationFormat}
分镜意图：${input.scene.summary || input.scene.title}
视觉方向：${visualPrompt}
```

其中 `visualPrompt` 的取值优先级为：

```text
input.scene.prompt
input.scene.b_roll_request
input.scene.summary
input.scene.title
```
